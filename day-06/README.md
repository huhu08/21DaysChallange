# Day 6: Week 1 Project - Calculator

## What Will We Build?
A calculator demonstrating Week 1 concepts:
- Variables and functions
- Control flow
- Ownership and borrowing

## Features
- Basic arithmetic (+, -, *, /)
- Advanced operations (power, sqrt, factorial)
- Calculator struct with history

## Run the Examples

```bash
cd day-06
cargo run
```

## Homework

### Homework 1: Add More Operations
Add percentage, absolute value, min/max functions.

### Homework 2: Interactive Calculator
Read operations from stdin.

### Homework 3: Memory Functions
Add memory_store, memory_recall, memory_clear to Calculator.
